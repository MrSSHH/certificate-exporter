# SQLite Certificate Exporter

A Tkinter-based GUI application to search for suppliers in an SQLite database, filter their certificates, and export related item data to Excel.

## 🚀 Features
* **Live Search:** Filter suppliers in real-time.
* **Dynamic Table Loading:** Map database tables via `tables.json`.
* **Excel Export:** Automatically generates formatted `.xlsx` files with Hebrew headers.
* **Progress Tracking:** Indeterminate progress bar for database queries.
